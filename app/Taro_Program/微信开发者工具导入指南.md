### 10. 常见问题

**Q: 编译失败怎么办?**
A: 重新运行 `npm run build:weapp`

**Q: 真机扫码后提示"该小程序已被删除"?**
A: 正常,使用的是测试AppID。实际部署需要注册小程序账号。

**Q: 如何修改代码?**
A: 
1. 修改 `src/` 目录下的源码
2. 运行 `npm run build:weapp` 重新编译
3. 或使用 `npm run dev:weapp` 开启watch模式

**Q: 页面报错怎么办?**
A: 打开微信开发者工具的"调试器"(Console)查看具体错误信息
