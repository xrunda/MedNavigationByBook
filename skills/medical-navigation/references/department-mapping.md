# 全科症状→科室映射规则（基于教材）

本文件为 medical-navigation 技能的科室推荐参考，覆盖教材（物理诊断学 诊断篇）全部 19 个常见症状。

---

## 一、急诊指征总表（所有症状通用）

### 🔴 立即急诊（危及生命，必须立即拨打120）

| 系统 | 急诊指征 |
|------|---------|
| **心血管** | 剧烈胸痛伴大汗/面色苍白/血压下降（疑似心梗）、严重心悸伴晕厥 |
| **呼吸** | 严重呼吸困难不能平卧、口唇发紫、大咯血(>100ml)、窒息感 |
| **腹部** | 剧烈腹痛伴板状腹（腹肌强直）、大量呕血或便血伴休克征 |
| **神经** | 突发意识丧失/昏迷、剧烈头痛伴颈强直、肢体瘫痪 |
| **儿科** | 热性惊厥持续>5分钟、意识障碍、严重脱水 |
| **其他** | 高热>41℃伴意识障碍、严重外伤、电击伤、溺水 |

### 🟡 紧急但非危重（24小时内就诊）

- 中度发烧（38-39°C）伴明显不适
- 腹痛持续>24小时无改善但无板状腹
- 胸痛但无典型心梗表现
- 呼吸困难但可平卧
- 便血但生命体征稳定

## 二、19 症状→科室映射

### 1. 发热

**常见科室映射**：

| 临床特征 | 推荐科室 | 紧急程度 | 说明 |
|---------|---------|---------|------|
| 感染性发热伴呼吸道症状 | 呼吸内科/感染科 | 🟡 | 咳嗽咳痰→呼吸；高热中毒→感染 |
| 感染性发热伴泌尿道症状 | 泌尿外科/肾内科 | 🟡 | 尿频尿急尿痛→泌尿 |
| 感染性发热伴消化道症状 | 消化内科/感染科 | 🟡 | 腹痛腹泻→消化 |
| 不明原因发热>2周 | 感染科 | 🟢 | 需系统排查 |
| 伴关节肿痛 | 风湿免疫科 | 🟢 | 结缔组织病？ |
| 伴出血倾向 | 血液科 | 🟡 | 血液系统疾病？ |
| 儿童发热 | 儿科 | 🟡/🟢 | 高热或精神差→急诊 |
| 高热>39℃伴意识障碍 | **急诊** | 🔴 | 立即就医 |

**决策要点**：
- 先判断热型（稽留/弛张/间歇）和伴随症状定位系统
- 儿童、老年人、孕妇急诊阈值放低

### 2. 胸痛

**常见科室映射**：

| 临床特征 | 推荐科室 | 紧急程度 | 说明 |
|---------|---------|---------|------|
| 压榨性胸痛>15分钟+大汗+放射痛 | **心内科/急诊** | 🔴 | 疑似急性心梗 |
| 撕裂样胸痛放射背部 | **心外科/急诊** | 🔴 | 疑似主动脉夹层 |
| 胸痛伴咳嗽咳痰发热 | 呼吸内科 | 🟡 | 肺炎、胸膜炎 |
| 胸骨后烧灼痛与进食相关 | 消化内科 | 🟢 | 胃食管反流 |
| 胸壁局部压痛 | 骨科/疼痛科 | 🟢 | 肋软骨炎、肌肉痛 |
| 伴呼吸困难但无心血管高危 | 呼吸内科 | 🟡 | 进一步排查 |

**决策要点**：
- 心源性 vs 肺源性 vs 骨骼肌肉 vs 消化道，鉴别至关重要
- 任何持续>15分钟的压榨性胸痛都必须急诊排除心梗

### 3. 腹痛

**常见科室映射**：

| 临床特征 | 推荐科室 | 紧急程度 | 说明 |
|---------|---------|---------|------|
| 转移性右下腹痛+反跳痛 | **普外科** | 🟡 | 疑似阑尾炎 |
| 上右腹痛+发热+油腻食物诱发 | **肝胆外科/普外科** | 🟡 | 疑似胆囊炎 |
| 上中腹/上左腹痛+烧灼感 | 消化内科 | 🟢 | 胃炎、胃溃疡 |
| 全腹痛+腹泻+不洁饮食 | 消化内科/感染科 | 🟢 | 急性胃肠炎 |
| 板状腹（腹肌强直） | **急诊** | 🔴 | 腹膜炎、穿孔 |
| 下腹痛（女性） | 妇产科/普外科 | 🟡 | 需排除宫外孕 |
| 伴血便+休克征 | **急诊** | 🔴 | 消化道大出血 |
| 儿童腹痛 | 儿科 | 🟡 | 肠套叠？阑尾炎？ |

**腹痛九宫格映射**（保留并优化）：

```
        左       中       右
上   │ 上左    上中     上右
中   │ 中左    中中     中右
下   │ 下左    下中     下右
```

| 位置 | 可能脏器 | 推荐科室 |
|------|---------|---------|
| 上左 | 胃、脾 | 消化内科、心内科 |
| 上中 | 胃、十二指肠 | 消化内科 |
| 上右 | 肝、胆囊 | 肝胆外科、消化内科 |
| 中左 | 结肠脾曲 | 消化内科、普外科 |
| 中中 | 小肠 | 消化内科、全科 |
| 中右 | 升结肠 | 普外科、消化内科 |
| 下左 | 乙状结肠、左侧输尿管 | 消化内科、泌尿外科 |
| 下中 | 膀胱、子宫/前列腺 | 消化内科、泌尿外科、妇科 |
| 下右 | **阑尾**、盲肠、右侧卵巢 | **普外科**、妇科 |

| 人群 | 处理原则 | 科室优先级 |
|------|---------|-----------|
| **儿童（<14岁）** | 优先儿科；注意肠套叠、热性惊厥等特殊疾病 | 儿科 > 专科 |
| **孕产妇** | 优先妇产科；异位妊娠为致命急症 | 妇产科 > 其他 |
| **老年人（>65岁）** | 注意不典型表现；急诊阈值放低 | 急诊/老年科 |
| **育龄女性下腹痛** | 必问月经史和妊娠可能 | 妇产科优先排查 |

## 四、常见疾病模式速查

### 急性阑尾炎
- **症状**：转移性右下腹痛 → 右下腹固定压痛 → 反跳痛
- **科室**：普外科
- **紧急程度**：🟡 尽快就诊

### 急性胆囊炎
- **症状**：右上腹痛 + 油腻食物诱发 + 发热
- **科室**：肝胆外科/普外科
- **紧急程度**：🟡 尽快就诊

### 急性心肌梗死
- **症状**：压榨性胸痛>15分钟 + 大汗 + 放射痛
- **科室**：**心内科/急诊**
- **紧急程度**：🔴 立即急诊

### 急性左心衰
- **症状**：端坐呼吸 + 粉红色泡沫痰 + 口唇发紫
- **科室**：**心内科/急诊**
- **紧急程度**：🔴 立即急诊

### 肺炎
- **症状**：发热 + 咳嗽 + 胸痛 + 气促
- **科室**：呼吸内科/感染科
- **紧急程度**：🟢/🟡 视严重程度

### 泌尿系感染
- **症状**：尿频尿急尿痛 + 可能发热腰痛
- **科室**：泌尿外科
- **紧急程度**：🟢 普通门诊

### 肾结石
- **症状**：腰痛剧烈放射腹股沟 + 血尿 + 辗转不安
- **科室**：泌尿外科
- **紧急程度**：🟡 尽快就诊

### 急性胃肠炎
- **症状**：腹痛腹泻 + 恶心呕吐 + 不洁饮食史
- **科室**：消化内科/感染科
- **紧急程度**：🟢/🟡 视脱水程度

### 蛛网膜下腔出血
- **症状**：突发"一生中最剧烈的头痛" + 颈强直
- **科室**：**神经外科/神经内科/急诊**
- **紧急程度**：🔴 立即急诊

## 使用说明

1. **急诊指征优先**：任何患者只要符合急诊指征，立即建议急诊，不再继续问诊
2. **症状组合判断**：单一症状可能对应多科室，需结合伴随症状综合判断
3. **特殊人群优先**：儿童、孕产妇、老年人优先考虑相应专科
4. **不确定时**：症状复杂或不典型 → 建议全科医学科或普通内科初步评估
5. **参考教材**：具体问诊维度见 `symptom-inquiry-guide.md`
