# 按主诉症状的问诊路径速查表

本文件供 medical-navigation 技能在问诊时使用。根据患者主诉匹配到教材症状后，按本表查找该症状在每一轮问诊中应重点询问的内容。

**使用方式**：第1轮确认主诉后 → 匹配到下方某个症状 → 按该症状的"问诊路径"逐轮询问。

---

### 1. 发热

**教材对应**：BOOK/IMG_1091-1093

**主诉识别关键词**："发烧"、"体温高"、"忽冷忽热"、"发热"

**问诊路径**：

| 轮次 | 重点问题 | 教材依据 |
|------|---------|---------|
| 第2轮（现病史） | - 体温最高多少度？<br>- 发热模式（突然升高/慢慢升高）<br>- 热型（一直烧/一阵一阵/早晚重）<br>- 有无畏寒、寒战？ | 临床表现：发热分度、体温上升期、临床过程 |
| 第3轮（加重/缓解） | - 什么情况下体温升高或降低？<br>- 用退烧药效果如何？<br>- 从开始到现在是加重还是好转？ | 问诊要点：用药与效果 |
| 第4轮（伴随症状） | - 全身：发冷、寒战、没力气？<br>- 呼吸道：咳嗽、嗓子疼、流鼻涕？<br>- 消化道：恶心、拉肚子、肚子疼？<br>- 泌尿：小便次数多、尿痛？<br>- 其他：皮疹、关节痛、淋巴结肿大？ | 伴随症状：寒战、皮疹、结膜充血、淋巴结肿大、肝脾大、出血、关节肿痛 |
| 第5轮（补充） | - 最近有没有去过人群密集的地方？<br>- 家里或同事有没有类似发烧的？<br>- 以前有没有类似发作？ | 问诊要点：流行病学史 |

**急诊警示征象**：
- 高热>41℃
- 高热伴意识障碍或精神恍惚
- 高热伴抽搐（儿童）
- 伴严重脱水（无尿、眼窝凹陷）

**关键鉴别点**：
- 热型（稽留热→伤寒/大叶性肺炎；弛张热→败血症；间歇热→疟疾）
- 伴随症状定位系统（呼吸道/消化道/泌尿道/神经系统）

### 2. 腹痛

**教材对应**：BOOK/IMG_1095

**主诉识别关键词**："肚子疼"、"胃疼"、"腹部不舒服"、"肚子痛"

**问诊路径**：

| 轮次 | 重点问题 | 教材依据 |
|------|---------|---------|
| 第2轮（现病史） | - 具体位置（上左/上中/上右/中左/中中/中右/下左/下中/下右）？<br>- 疼痛性质（隐痛/胀痛/绞痛/刺痛/烧灼感）？<br>- 程度（轻微/中等/剧烈）？<br>- 持续还是阵发？疼了多久？ | 临床表现：部位、性质与程度、发作时间 |
| 第3轮（加重/缓解） | - 与进食关系（空腹重还是饭后重）？<br>- 与体位关系（躺着/坐着/弯腰）？<br>- 排便后是否缓解？<br>- 是越来越重还是有好转？ | 临床表现：诱发因素、与体位关系 |
| 第4轮（伴随症状） | - 全身：发烧、发冷？<br>- 消化：恶心、想吐、拉肚子、便秘、大便带血或发黑？<br>- 泌尿：小便次数多、尿急、尿痛、尿血？<br>- 其他：最近吃什么了？以前有没有类似肚子疼？ | 伴随症状：发热寒战、恶心呕吐、腹泻便秘便血、尿频尿急尿血 |
| 第5轮（补充） | - 女性：上次月经什么时候？有没有可能怀孕？<br>- 既往：有没有做过腹部手术？有没有胃溃疡/胆结石病史？ | 既往史、特殊人群（女性月经孕产） |

**急诊警示征象**：
- 剧烈腹痛伴板状腹（腹肌硬如木板）
- 剧烈腹痛伴高热>39℃
- 腹痛伴大量呕血或便血
- 腹痛伴休克（头晕、出冷汗、心慌）
- 孕妇腹痛伴阴道出血

**关键鉴别点**：
- 内脏性痛（弥散、定位模糊）vs 躯体性痛（定位准确、反跳痛）vs 牵涉痛
- 转移性右下腹痛→阑尾炎
- 上右腹痛+油腻食物→胆囊炎

### 3. 胸痛

**教材对应**：BOOK/IMG_1094、表1-2

**主诉识别关键词**："胸口疼"、"心口痛"、"胸闷"、"胸痛"

**问诊路径**：

| 轮次 | 重点问题 | 教材依据 |
|------|---------|---------|
| 第2轮（现病史） | - 具体位置（胸骨后/左侧/右侧/心前区）？<br>- 性质（压榨感/刺痛/烧灼感/撕裂感）？<br>- 持续多久（几秒/几分钟/几小时）？<br>- 向左肩、左臂、后背、下巴放射？ | 临床表现：部位、性质、持续时间、放射痛 |
| 第3轮（加重/缓解） | - 与活动关系（走路/上楼梯时加重）？<br>- 与情绪关系（生气/紧张时）？<br>- 休息或含服硝酸甘油是否缓解？<br>- 与呼吸/体位关系（深呼吸/翻身时疼）？ | 临床表现：影响因素（劳力/情绪/呼吸/体位） |
| 第4轮（伴随症状） | - 呼吸：咳嗽、咳痰、喘不上气？<br>- 循环：心慌、心跳快、出冷汗、面色苍白？<br>- 消化：吞咽困难、烧心反酸？<br>- 其他：最近有没有感冒？ | 伴随症状：伴咳嗽咳痰发热、伴呼吸困难、伴咯血、伴苍白大汗血压下降、伴吞咽困难 |
| 第5轮（补充） | - 年龄多大？有没有高血压、糖尿病、高血脂？<br>- 有没有吸烟？家里有没有心脏病史？ | 既往史（心血管危险因素） |

**急诊警示征象**：
- 剧烈压榨性胸痛>15分钟伴大汗
- 胸痛伴晕厥或意识丧失
- 胸痛伴严重呼吸困难、口唇发紫
- 撕裂样胸痛放射背部（疑似主动脉夹层）

**关键鉴别点**：
- 心绞痛：压榨感、活动诱发、休息/硝酸酯缓解、持续1-5分钟
- 心梗：持续>15分钟、伴大汗、硝酸酯不缓解
- 胸膜炎：与呼吸相关、刺痛

### 4. 头痛与头晕

**教材对应**：BOOK/IMG_1096-1097

**主诉识别关键词**："头疼"、"头昏"、"头晕"、"眩晕"、"头胀"

**问诊路径**：

| 轮次 | 重点问题 | 教材依据 |
|------|---------|---------|
| 第2轮（现病史） | - 部位（额头/太阳穴/后脑勺/全头）？单侧还是双侧？<br>- 性质（一跳一跳/紧箍感/电击样/胀痛）？<br>- 程度（轻微/中等/剧烈）？<br>- 是头晕还是天旋地转（眩晕）？ | 临床表现：部位、程度与性质 |
| 第3轮（加重/缓解） | - 与咳嗽/打喷嚏/用力关系？<br>- 与体位关系（躺下/坐起/低头）？<br>- 与光线/声音关系（怕光/怕吵）？<br>- 活动后是否加重？ | 临床表现：加重/缓解因素 |
| 第4轮（伴随症状） | - 全身：发烧、恶心呕吐？<br>- 神经：视力模糊、手脚麻、说话不利索？<br>- 颈部：脖子硬、脖子疼？<br>- 耳朵：耳鸣、耳朵闷、听力下降？ | 伴随症状：发热、恶心呕吐、视力障碍、神经体征、颈强 |
| 第5轮（补充） | - 最近有没有头部受伤？<br>- 有没有高血压、糖尿病？<br>- 以前有没有类似头痛？ | 既往史、外伤史 |

**急诊警示征象**：
- 突发"一生中最剧烈的头痛"
- 剧烈头痛伴颈强直
- 头痛伴意识障碍、抽搐、偏瘫
- 头痛伴高热、颈强直（疑似脑膜炎）

**关键鉴别点**：
- 偏头痛：反复发作、搏动性、伴恶心呕吐畏光、活动加重
- 紧张型头痛：双侧紧箍感、与情绪紧张相关
- 继发性头痛：突发剧烈、伴神经体征、需排除颅内病变

### 5. 咳嗽与咳痰

**教材对应**：BOOK/IMG_1097-1098

**主诉识别关键词**："咳嗽"、"咳痰"、"嗓子痒想咳"、"干咳"

**问诊路径**：

| 轮次 | 重点问题 | 教材依据 |
|------|---------|---------|
| 第2轮（现病史） | - 干咳还是有痰？<br>- 病程多长（几天/几周/几个月）？<br>- 时间规律（晨起/夜间/全天）？<br>- 音色（嘶哑/金属音/犬吠样）？ | 临床表现：咳嗽性质、时间与规律、音色 |
| 第3轮（加重/缓解） | - 什么情况下加重（冷空气/灰尘/说话）？<br>- 与体位关系（躺下加重）？<br>- 用药史（有没有吃降压药如ACEI）？ | 问诊要点：用药史、环境因素 |
| 第4轮（伴随症状） | - 全身：发烧、没力气、消瘦？<br>- 呼吸：喘不上气、胸痛、嗓子有呼噜声（哮鸣音）？<br>- 痰：痰量多少？什么颜色（白/黄/绿/铁锈色）？有没有血？<br>- 其他：手指末端是不是变粗了（杵状指）？ | 伴随症状：伴发热、伴胸痛、伴呼吸困难、伴哮鸣音、伴杵状指 |
| 第5轮（补充） | - 有没有抽烟？抽了多少年？<br>- 有没有哮喘、结核病史？<br>- 工作环境有没有粉尘？ | 既往史、吸烟史、环境暴露 |

**急诊警示征象**：
- 咳嗽伴大咯血
- 咳嗽伴严重呼吸困难、口唇发紫
- 咳嗽伴意识障碍

**关键鉴别点**：
- 病程：急性（<3周）→感染；亚急性（3-8周）→感染后；慢性（>8周）→需系统排查
- 痰色：白色→病毒/过敏；黄绿→细菌感染；铁锈色→肺炎球菌；粉红泡沫→心衰

### 6. 呼吸困难

**教材对应**：BOOK/IMG_1099

**主诉识别关键词**："喘不上气"、"呼吸困难"、"气短"、"胸闷"、"憋气"

**问诊路径**：

| 轮次 | 重点问题 | 教材依据 |
|------|---------|---------|
| 第2轮（现病史） | - 吸气困难、呼气困难，还是都有？<br>- 严重程度（活动后/安静时/不能平躺）？<br>- 起病急缓（突然/逐渐）？<br>- 能不能平卧睡觉？ | 发生机制及临床表现：肺源性（吸气性/呼气性/混合性） |
| 第3轮（加重/缓解） | - 与活动关系（走几步就喘）？<br>- 与体位关系（坐着好点/躺下加重）？<br>- 夜间会不会憋醒（端坐呼吸）？<br>- 是越来越重还是有好转？ | 心源性：端坐呼吸、夜间阵发性呼吸困难 |
| 第4轮（伴随症状） | - 呼吸：咳嗽、咳痰（什么颜色）、嗓子呼噜声（哮鸣音）？<br>- 循环：心慌、胸痛、下肢水肿？<br>- 全身：发烧、嘴唇或手指发紫？<br>- 其他：最近有没有感冒？有没有心脏病/肺病？ | 伴随症状：伴哮鸣音、伴发热、伴单侧胸痛、伴咳嗽咳痰 |
| 第5轮（补充） | - 有没有心脏病、高血压、糖尿病？<br>- 有没有肾病、肺病？<br>- 有没有接触什么毒物/药物？ | 诱因/基础病 |

**急诊警示征象**：
- 严重呼吸困难不能平卧、口唇发紫
- 端坐呼吸+粉红色泡沫痰（急性左心衰）
- 吸气性呼吸困难+喉鸣（上气道梗阻）
- 呼吸困难伴意识障碍

**关键鉴别点**：
- 肺源性：吸气性（上气道）、呼气性（哮喘/COPD）、混合性（肺炎/肺纤维化）
- 心源性：端坐呼吸、夜间阵发、伴水肿
- 中毒性：代谢性酸中毒深大呼吸

### 7. 咯血

**教材对应**：BOOK/IMG_1100、表1-3

**主诉识别关键词**："咳血"、"痰里有血"、"吐血"（注意鉴别是咯血还是呕血）

**问诊路径**：

| 轮次 | 重点问题 | 教材依据 |
|------|---------|---------|
| 第2轮（现病史） | - 咯血量（痰中带血/满口血/大量）？<br>- 颜色（鲜红/暗红/铁锈色）？<br>- 混有痰还是泡沫？<br>- 是咳出来的还是呕出来的？ | 临床表现：咯血量与性状 |
| 第3轮（加重/缓解） | - 什么情况下出现（咳嗽后/无明显诱因）？<br>- 是第一次还是反复发生？<br>- 咯血前有没有喉痒、胸闷、咳嗽？ | 与呕血鉴别：出血前喉痒/胸闷咳嗽 vs 上腹不适恶心呕吐 |
| 第4轮（伴随症状） | - 全身：发烧、盗汗、没力气、消瘦？<br>- 呼吸：咳嗽、咳痰（脓痰？）、胸痛、喘不上气？<br>- 皮肤：皮肤或牙龈出血？<br>- 其他：年龄多大？抽不抽烟？ | 伴随症状：伴发热、伴胸痛、伴脓痰、伴呛咳、伴皮肤黏膜出血、伴消瘦 |
| 第5轮（补充） | - 有没有结核病史？<br>- 家里有没有结核病人？<br>- 有没有心脏病（二尖瓣狭窄）？ | 既往史、流行病学史 |

**急诊警示征象**：
- 大咯血（>100ml）或持续咯血不止
- 咯血伴严重呼吸困难
- 咯血伴休克征象

**关键鉴别点**：
- 咯血 vs 呕血：出血前感觉（喉痒/胸闷 vs 上腹不适）、出血方式（咳出 vs 呕出）、颜色（鲜红 vs 咖啡）、混有物（痰泡 vs 食物残渣）
- 病因：青年+结核史→结核；中年+吸烟→肺癌；脓痰→支扩

### 8. 发绀

**教材对应**：BOOK/IMG_1100-1101

**主诉识别关键词**："嘴唇发紫"、"手脚发紫"、"发绀"、"青紫"

**问诊路径**：

| 轮次 | 重点问题 | 教材依据 |
|------|---------|---------|
| 第2轮（现病史） | - 发绀部位（嘴唇/手指/脚趾/全身）？<br>- 皮肤温度（发凉还是温暖）？<br>- 按摩或保暖后是否改善？<br>- 起病时间（自幼/近期）？ | 临床表现：中心性 vs 周围性发绀 |
| 第3轮（加重/缓解） | - 什么情况下加重（活动/寒冷）？<br>- 休息后是否缓解？<br>- 是越来越重还是稳定的？ | 与活动关系 |
| 第4轮（伴随症状） | - 呼吸：喘不上气、咳嗽？<br>- 循环：心慌、心跳快、下肢水肿？<br>- 其他：手指末端是不是膨大（杵状指）？ | 伴随症状：呼吸困难、杵状指、意识障碍 |
| 第5轮（补充） | - 有没有心脏病、肺病？<br>- 平时吸不吸烟？<br>- 工作环境有没有接触什么化学物质？ | 既往史、职业暴露 |

**急诊警示征象**：
- 发绀伴严重呼吸困难
- 发绀伴意识障碍
- 新生儿发绀

**关键鉴别点**：
- 中心性发绀（全身、黏膜、温暖）→心肺疾病
- 周围性发绀（肢端、发凉、按摩缓解）→周围血管/循环问题

### 9. 心悸

**教材对应**：BOOK/IMG_1101

**主诉识别关键词**："心慌"、"心跳快"、"心悸"、"心脏跳得厉害"、"心脏漏跳"

**问诊路径**：

| 轮次 | 重点问题 | 教材依据 |
|------|---------|---------|
| 第2轮（现病史） | - 什么感觉（心慌/漏跳/心跳剧烈/心跳不齐）？<br>- 持续时间（几秒/几分钟/几小时）？<br>- 发作频率（偶尔/经常/持续）？<br>- 心率大概多快（能数出来吗）？ | 临床表现：心悸感觉、持续时间、发作频率 |
| 第3轮（加重/缓解） | - 诱因（情绪紧张/运动/喝酒/咖啡/浓茶）？<br>- 缓解方式（休息/深呼吸/按压眼球）？<br>- 与体位关系（躺着/坐着）？ | 诱因、缓解方式、体位影响 |
| 第4轮（伴随症状） | - 循环：胸痛、头晕、眼前发黑、晕厥？<br>- 全身：手抖、消瘦、怕热（甲亢症状）？<br>- 其他：出汗、焦虑、濒死感？ | 伴随症状：胸痛、晕厥、出汗、手抖消瘦 |
| 第5轮（补充） | - 家里有没有心脏病史？<br>- 以前有没有类似发作？做过心电图吗？<br>- 最近在吃什么药？ | 家族史、既往史、用药史 |

**急诊警示征象**：
- 心悸伴晕厥
- 心悸伴胸痛
- 心悸伴严重呼吸困难
- 持续快速心率>150次/分

**关键鉴别点**：
- 生理性：情绪激动、运动、烟酒咖啡
- 病理性：心律失常、甲亢、贫血、低血糖
- 心理性：焦虑、心脏神经症

### 10. 黄疸

**教材对应**：BOOK/IMG_1102-1103、表1-4

|------|---------|---------|
| 第2轮（现病史） | - 孩子多大年龄？<br>- 发烧和抽筋哪个在先？体温多少度？<br>- 抽筋是什么表现（全身抽/一侧抽/眼睛上翻/口吐白沫）？<br>- 持续了多长时间？<br>- 24小时内抽了几次？ | 定义与分类：单纯型 vs 复杂型；发作形式 |
| 第3轮（加重/缓解） | - 发作前有没有咳嗽、拉肚子等感染征象？<br>- 抽完后多长时间清醒？<br>- 清醒后精神状态怎么样？ | 感染触发、惊厥后状态 |
| 第4轮（伴随症状） | - 抽的时候有没有嘴唇发紫？<br>- 抽完后有没有一侧手脚没力气（Todd麻痹）？<br>- 有没有呕吐（喷射性？）、脖子硬？<br>- 以前有没有这样抽过？ | 伴随症状：感染征象、Todd麻痹、脑膜刺激征 |
| 第5轮（补充） | - 出生时有没有缺氧、难产？<br>- 孩子发育怎么样（会不会走路/说话）？<br>- 家里有没有其他人小时候发烧抽过？<br>- 父母有没有癫痫病史？ | 围产史、发育、家族史 |

**急诊警示征象**：
- 惊厥持续>5分钟
- 复杂型（局灶性发作、>15分钟、24小时内≥2次）
- 惊厥后意识不恢复
- 伴高热颈强直（排除脑膜炎）

**关键鉴别点**：
- 单纯型：全身性、<15分钟、24h内1次、热退止惊、无后遗症
- 复杂型：局灶性、>15分钟、24h内≥2次、需排除其他病因
- 与颅内感染鉴别：有无颈强直、意识障碍、病理征

## 使用说明

1. **快速定位**：第1轮确认主诉后，立即在本表中找到对应症状
2. **逐轮询问**：严格按照"第2轮→第3轮→第4轮→第5轮"的顺序询问
3. **急诊优先**：任意轮次发现"急诊警示征象"，立即中断并建议急诊
4. **通俗用语**：问诊时使用患者能懂的语言（如"拉肚子"而非"腹泻"）
5. **合并提问**：每轮尽量合并 3-5 个问题一起问，控制总轮数
6. **参考教材**：具体机制、病因、临床表现详见 BOOK/ 扫描页和 `../diagnostic-reasoning/references/inquiry-checklist.md`
